## ğŸ“‹ Deployment Steps

### Step 1: Go to Render.com
1. Open your browser
2. Go to: **https://render.com**
3. Click **"Get Started"** or **"Sign In"**

### Step 2: Sign Up/Login
- **Option A**: Sign in with GitHub (Recommended)
- **Option B**: Sign in with GitLab
- **Option C**: Sign in with Email

**âœ¨ Choose GitHub** - It's the easiest!

### Step 3: Create New Web Service
1. Once logged in, click **"New +"** button (top right)
2. Select **"Web Service"** from the dropdown
3. You'll see a page to connect your repository

### Step 4: Connect Your GitHub Repository
1. Click **"Connect account"** if not already connected
2. Authorize Render to access your GitHub
3. You'll see a list of your repositories
4. Find and click: **"Connect"** next to `FileSharingSystem`

### Step 5: Configure Your Service

Render will auto-detect your settings from `render.yaml`, but verify:

**Basic Settings:**
- **Name**: `blockchain-file-sharing` (or any name you like)
- **Region**: Choose closest to you (e.g., Oregon, Frankfurt, Singapore)
- **Branch**: `main`
- **Root Directory**: Leave empty

**Build Settings:**
- **Build Command**: `pip install -r requirements.txt`
- **Start Command**: `gunicorn app:app --bind 0.0.0.0:$PORT`

**Environment Variables** (Auto-configured from render.yaml):
- `FLASK_ENV` = `production`
- `SECRET_KEY` = (Auto-generated by Render)
- `DEBUG` = `False`
- `BLOCKCHAIN_DIFFICULTY` = `2`
- `HOST` = `0.0.0.0`

### Step 6: Choose Your Plan
- Select **"Free"** plan
- 750 hours/month free
- 512 MB RAM

### Step 7: Deploy!
1. Click **"Create Web Service"** button
2. Render will start building your app
3. Watch the build logs in real-time

## ğŸ”§ Post-Deployment Configuration

### Update Frontend API URL

After deployment, update your JavaScript to use the production URL:

**File**: `static/script_advanced.js`

Change line 2:
```javascript
const API_BASE_URL = 'https://your-app-name.onrender.com/api';
```

Replace `your-app-name` with your actual Render URL.

Then commit and push:
```bash
git add static/script_advanced.js
git commit -m "Update API URL for production"
git push origin main
```

Render will **auto-deploy** the changes!

## ğŸ› Troubleshooting

### Build Failed?

**Check Python version:**
```yaml
# In render.yaml
- key: PYTHON_VERSION
  value: 3.13.0
```

**Check requirements.txt:**
```bash
# Make sure all dependencies are listed
Flask==3.0.0
gunicorn==21.2.0
cryptography==41.0.7
```

### App Not Starting?

**Check start command:**
```
gunicorn app:app --bind 0.0.0.0:$PORT
```

**View logs** in Render dashboard for error details.

### 502 Bad Gateway?

This is normal during initial startup. Wait 30-60 seconds.
